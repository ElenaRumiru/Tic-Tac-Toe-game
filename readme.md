## About

Two-player Tic Tac Toe with a stunning **neon glow** aesthetic. Hand-drawn glowing X's and O's bring a fresh visual spin to the classic 3x3 game. Green neon for X, pink neon for O — played in a dark, atmospheric setting.

## How to Play

1. **Player X** goes first (green neon)
2. Click any empty cell to place your mark
3. **Player O** responds (pink neon)
4. First to get **3 in a row** wins!
5. Hit **Restart** for a rematch

## Features

- Two-player local multiplayer on a 3x3 grid
- Neon glow visual design with hand-drawn marks
- Win detection (rows, columns, diagonals)
- Draw detection
- Custom neon font (Shock Brat)
- Fully responsive — works on mobile and desktop
